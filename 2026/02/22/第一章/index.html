  2. 因为任意一个圆盘有三种可能的摆放，所以 n 个圆盘有 $3^n$ 种摆放方式。
    最短的移动序列有 $3^n-1$ 步，且不会有任意两步后的状态完全相同，所以共有 $3^n$ 种状态
  3. 不能。

    所以第 $4$ 个圆最多再分割出 $2 * (4 - 1) = 6$ 个区间
  4. $Q_{0}=\alpha$，
    $Q_{1}=\beta$，
    $Q_{2}=\dfrac{1+\beta}\alpha$，
    $Q_{3}=\dfrac{1+\alpha+\beta}{\alpha\beta}$，
    $Q_{4}=\dfrac{1+\alpha}\beta$。
    得到规律
    $Q_{5k}=\alpha$，
    $Q_{5k+1}=\beta$，
    $Q_{5k+2}=\dfrac{1+\beta}\alpha$，
    $Q_{5k+3}=\dfrac{1+\alpha+\beta}{\alpha\beta}$，
    $Q_{5k+4}=\dfrac{1+\alpha}\beta$。
  5. a.
    $x_{1}…x_{n}=x_{1}…x_{n-1}[(x_{1}+…+x_{n-1})/(n-1)]$
    $[(x_{1}+…+x_{n})/n]^n=(x_{1}+…+x_{n-1}+x_{n})^n=[(x_{1}+…+x_{n-1})/(n-1)]^(n-1)    [(x_{1}+…+x_{n-1})/(n-1)]$
    所以 $x_{1}…x_{n}<=[(x_{1}+…+x_{n})/n]^n$ 可化为 $x_{1}…x_{n-1}*[(x_{1}+…+x_{n-1})/(n-1)]<=[(x_{1}+…+x_{n-1})/(n-1)]^(n-1)*[(x_{1}+…+x_{n-1})/(n-1)]$
    又因为 $[(x_{1}+…+x_{n-1})/(n-1)]>=0$ 所以 $x_{1}…x_{n-1}<=[(x_{1}+…+x_{n-1})/(n-1)]^(n-1)$
    即为 P(n) 蕴含 P(n-1)
    b.
    P(2n) ：x_1 * x_2 … * x_2n-1 * x_2n = ((x_1 + x_2 + … + x_2n-1 + x_2n) / 2n) ^ 2n
    由 P(n) 得 x_1 * x_2 … * x_2n-1 * x_2n <= ((x_1 + … + x_n) / n) ^ n * ((x_n+1 + … + x_2n) / n) ^ n
    由 P(2) 得 ((x_1 + … + x_n) / n) ^ n * ((x_n+1 + … + x_2n) / n) ^ n <= ((x_1 + … + x_n + x_{n+1} + … + x{2n}) / n / 2) ^ 2 ^ n = ((x_1 + … + x_n + x_{n+1} + … + x{2n}) / 2n) ^ 2n
    c.
    由 P(2) 可以推到 P(4)
    由 P(4) 可以推到 P(3)
    P(2k-1) 可以由 P(2k) 推出
    P(2k) 可以由 P(k) 推出
    所以 P(n) 对所有 n 为真
