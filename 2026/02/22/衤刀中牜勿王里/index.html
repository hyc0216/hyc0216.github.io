衤刀 中 牛勿王里

电与磁

电荷

同性相斥，异性相吸
波斯橡皮，波正橡负

电流

电流： I
单位： 安培 A
电荷的定向移动形成电流
I = Q / t
Q = 通过导体横截面的电荷量
1 A的电流表示 1s 内通过导体横截面的电荷量是1C
电流表：串联、正进负出
形成持续电流的条件：电压、通路

电压

电压：U
单位：伏特 V
一节干电池 1.5V
我国家庭电路电压 220V
人体安全电压 36V
电压表：并联、抱谁测谁、正进负出
相当于断路（但不是）

电阻

电阻：R
单位：欧姆 Ω
和导体长度、横截面积、材料、温度有关
滑动变阻器：串联使用，闭合开关前滑片移到阻值最大的位置（保护电路）

电路

I = U / R
串联电路：I = I1 = I2 U = U1 + U2 –> R = R1 + R2 U1 / R1 = U2 / R2
串联电路中阻值变大电压也变大
并联电路：I = I1 + I2 U = U1 = U2 –> 1/R = 1/R1 + 1/R2 = R1R2 / R1 + R2 I1/I2 = R2/R1
动态电路中电源电压永远不变

电能、电功

电能、电功 W
单位 焦耳 J
功率 P
单位 瓦特 W
W = UIt
P = W / t
1 KW*h = 3.6 * 1e6J
电路中 P = P1 + P2
额定电压、功率：电器正常工作时的电压、功率
灯泡亮度由实际功率决定U越大P越大

焦耳定律

Q = I^2Rt
Q = 电流通过导体产生的热量
纯电阻电路中 Q = W

家庭电路

左中右相上接地
我国家庭电路电压 220V
电流过大的原因：总功率过大或短路
灯泡脑袋接相线：防止拧灯泡触电

磁

外部磁感线从N极出发绕一圈回到S极

电生磁

安培定则（右手螺旋定则）用右手握住通电螺线管，让四指指向电流的方向，那么大拇指所指的那一端是通电螺线管的N极
电流越大、线圈匝数越密磁性越强

电动机

电能转化为机械能
当电流的方向或者磁感线的方向变得相反时， 通电导线受力的方向也变得相反

电磁感应

闭合电路的一部分导体在磁场中做切割磁感线运动事，导体中产生电流
机械能转电能
