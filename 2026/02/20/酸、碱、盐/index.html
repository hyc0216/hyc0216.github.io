酸、碱、盐

口诀

钾钠铵盐皆可溶
硝酸盐类性相同
氯化物中银不溶
硫酸盐中钡不溶
碳酸不溶钙钡银
铁铜镁铝碱不溶
硫酸钙银石灰水
不能除杂能沉淀

酸

定义

酸是指在水溶液中电离时，产生的阳离子全部都是H+的化合物
酸 = H+ + 酸根离子

常见的酸

强酸

盐酸 HCl、硫酸 H2SO4、硝酸 HNO3、高氯酸 HClO4
氢溴酸 HBr 氢碘酸 HI

弱酸：

以B为中心元素
硼酸 H3BO3
以C为中心元素
碳酸 H2CO3
醋酸 CH3COOH
草酸 H2C2O4
以N为中心元素
亚硝酸 HNO2
以F为中心元素
氢氟酸 HF
以Si为中心元素
硅酸 H2SiO3
以P为中心元素
磷酸 H3PO4
亚磷酸 H3PO3 二元酸
次磷酸 H3PO2 一元酸
以S为中心元素
亚硫酸 H2SO3
硫化氢 H2S
以Cl为中心元素
次氯酸 HClO
其他：
氢氰酸 HCN

性质

  1. 使紫色石蕊试液变红，使无色酚酞试液不变色。
  2. 与氢前金属发生置换反应
    如：Mg + 2HCl == MgCl2 + H2↑
    Fe + H2SO4 == FeSO4 + H2↑ 注意：生成的是硫酸亚铁哦，是浅绿色的
  3. 与碱发生中和反应
    酸 + 碱 == 盐 + 水
    e.g.
    HCl + NaOH == NaCl + H2O
    H2SO4 + 2NaOH == Na2SO4 + 2H2O
    2HCl + Ca(OH)2 == CaCl2 + 2H2O
  4. 与碱性氧化物反应
    酸 + 碱性氧化物 == 盐 + 水
    e.g. 2HCl +CuO = CuCl2 + H2O
  5. 与部分盐发生复分解反应
    反应条件：生成物中必须有沉淀、气体或水。
    酸 + 盐 == 新酸 + 新盐
    e.g. H2SO4 + BaCl2 = 2HCl + BaSO4↓
    CaCO3 + 2HCl == CaCl2 + H2O + CO2↑

碱

定义

碱是指在水溶液中电离时，产生的阴离子全部都是OH-的化合物
碱 = 金属离子 + OH-

常见的碱

强碱

氢氧化钾 KOH、氢氧化钠 NaOH、氢氧化钙 Ca(OH)2、氢氧化钡 Ba(OH)2

弱碱

氢氧化铜 Cu(OH)2 氢氧化铁 Fe(OH)3 （基本都是难溶性碱）
氨水/一水合氨 NH3·H2O （唯一常见可溶性碱）

性质

  1. 使紫色石蕊试液变蓝，使无色酚酞试液变红。
  2. 与酸发生中和反应
    酸 + 碱 == 盐 + 水
    e.g.
    HCl + NaOH == NaCl + H2O
    H2SO4 + 2NaOH == Na2SO4 + 2H2O
    2HCl + Ca(OH)2 == CaCl2 + 2H2O
  3. 与酸性氧化物反应
    碱 + 酸性氧化物 == 盐 + 水
    e.g.
    Ca(OH)2 + CO2 == CaCO3↓ + H2O
  4. 与部分盐发生复分解反应
    反应条件：生成物中必须有沉淀、气体或水。
    碱 + 盐 == 新碱 + 新盐
    e.g.
    2NaOH+ CuSO4 = Cu(OH)2↓ + Na2SO4
    Ba(OH)2 + CuSO4 == Cu(OH)2↓ + BaSO4↓
