四大基本反应类型

化合反应

A + B + …… –> X
e.g. CO2 + C == 2CO

分解反应

X –> A + B + ……
加热
e.g. KMnO4 ==== K2MnO4 + MnO2 + O2↑
H2CO3 == CO2↑ + H2O

置换反应

A + BC –> B + AC
e.g. Fe + 2HCl == FeCl2 + H2↑
Mg + 2HCl == MgCl2 + H2↑

复分解反应

AB + CD == AD + BC

  1. 反应物都是化合物

  2. 在反应中反应物相互交换成分

  3. 生成物中有沉淀或气体或水

  4. 酸 + 碱 –> 盐 + 水 反应物至少一种可溶

  5. 酸 + 盐 –> 新酸 + 新盐 反应物中的酸必须可溶

  6. 碱 + 盐 –> 新碱 + 新盐 反应物都必须可溶

  7. 盐1 + 盐2 –> 新盐1 + 新盐2 反应物都必须可溶

  8. 酸 + 金属氧化物 –> 盐 + 水

e.g.
Na2CO3 + Ca(OH)2 == CaCO3↓ + 2NaOH
NaOH + CuSO4 == Cu(OH)2↓ + Na2SO4
HCl + NaOH == NaCl + H2O （中和反应）

听大佬说是酸碱中和反应
Ca(OH)2 + CO2 == CaCO3↓ + H2O
2NaOH + CO2 == Na2CO3 + H2O
